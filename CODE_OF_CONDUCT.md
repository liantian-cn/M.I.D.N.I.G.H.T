# Code of Conduct

## Our Standard

This project expects respectful, factual, and constructive collaboration.

Unacceptable behavior includes:

1. Personal attacks or harassment.
2. Discriminatory language or hate speech.
3. Deliberate disruption, spam, or bad-faith arguments.
4. Requests or contributions aimed at unfair gameplay abuse.

## Expected Behavior

1. Focus on technical facts and reproducible evidence.
2. Critique ideas, not people.
3. Keep discussions concise and actionable.

## Enforcement

The maintainer may remove comments, close threads, reject contributions, or block participants that violate this code.

## Reporting

Please report serious conduct issues through repository issues with clear context.
