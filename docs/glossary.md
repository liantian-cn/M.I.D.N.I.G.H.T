# 术语表

## 核心术语

| 术语 | 英文 | 定义 |
|---|---|---|
| 矩阵 | Matrix | 屏幕中的固定像素编码区域，用于承载协议数据。 |
| 单元 | Cell | `4x4` 的基础像素编码块。 |
| 大单元 | MegaCell | `8x8` 的图标级编码块。 |
| 角标单元 | BadgeCell | 带状态角标扩展语义的单元结构。 |
| 绘制端 | Producer | 负责把状态渲染为像素的 DejaVu 端。 |
| 解析端 | Consumer | 负责截图采样与解码的 Terminal 端。 |

## 命名约束

1. 本仓库优先使用 `Cell/MegaCell/BadgeCell` 作为标准词。
2. 不同文档中出现同一术语时，含义必须一致。
3. 若术语含义变更，必须同步更新本文件与 `CHANGELOG.md`。
